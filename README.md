# つづくん 🌱 — 怠惰な人のための習慣化アプリ

「やるべきことが続かない」のは、意志が弱いからじゃない。**続く仕組み**が無いだけ。
つづくんは、行動科学で実証された方法を「摩擦ゼロ・自己否定ゼロ」のUIに落とし込んだ習慣化アプリです。アカウント不要・ログイン不要・データは端末内だけ。スマホにインストールして使えます（PWA）。

## なにが「科学的」なのか

| 仕組み | 元になっている考え方 | アプリでの実装 |
|---|---|---|
| 👣 極小ステップ | 2分ルール / Tiny Habits | 登録時に「これだけやればOK」な極小版を必ず設定 |
| 🪝 きっかけ（if-then） | 実装意図 / 習慣の積み重ね | 「〜したら、〜する」を設定。既存の習慣にくっつける |
| 🔥 連続記録 | Don't break the chain | ストリーク表示＋カレンダーのヒートマップ |
| 🎉 即時ごほうび | ドーパミン報酬 | ワンタップ達成で紙吹雪＋ポイント＋励まし |
| ❄️ まあいっか | 自己への思いやり / 全か無か思考の回避 | サボった日も連続を守れるフリーズ機能 |
| 🛋️ 低摩擦 | 行動のハードルを下げる | ログイン無し・大きなタップボタンだけ |

## 機能

- 今日やる習慣をワンタップで達成 → 演出とポイント
- 習慣の登録（テンプレ・絵文字・色・曜日・リマインド時刻）
- ストリーク／レベル・称号／30日達成率／ヒートマップ
- 通知リマインド（アプリ起動中）
- JSONでバックアップ書き出し・読み込み
- 初回オンボーディング

## 開発

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 本番ビルド（dist/）
npm run preview  # ビルド結果をローカル確認
```

技術スタック: **Vite + React + TypeScript + Tailwind CSS + zustand + vite-plugin-pwa**
データは `localStorage`（キー: `syukan-app-v1`）に保存。サーバー無し。

## GitHub Pages へ公開する

1. このリポジトリを GitHub に push する。
2. GitHub の **Settings → Pages → Build and deployment → Source** を **GitHub Actions** に設定。
3. `master`（または `main`）に push すると `.github/workflows/deploy.yml` が自動でビルド＆公開。
4. 公開URL（`https://<ユーザー名>.github.io/<リポジトリ名>/`）をスマホで開き、ブラウザの「ホーム画面に追加」でアプリとしてインストール。

`vite.config.ts` の `base: "./"` によりリポジトリ名に関係なくサブパス配信で動きます。

## データの保存場所

すべて端末のブラウザ内（localStorage）にのみ保存され、外部には送信されません。
機種変更や別端末への移行は、設定タブの「バックアップを書き出す／読み込む」で行えます。
